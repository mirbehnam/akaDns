🧠 ابزار تنظیم آسان DNS برای رفع تحریم ایران

با چند کلیک، تحریم‌های برنامه‌نویسی، هوش مصنوعی و گیمینگ را دور بزنید — بدون نیاز به VPN! این ابزار با استفاده از چندین دی ان اس تحریم شکن کار میکند و همینطور مشکل کار نکردن دی ان اس ها را رفع می کند پیشنهاد می شود حتما اموزش یوتوب را ببینید

آموزش کامل در یوتیوب: مشاهده آموزش

ویژگی‌ها

  • ✅ تنظیم آسان و خودکار DNSهای مخصوص دور زدن تحریم
  • 🖥️ رابط گرافیکی (GUI) برای راحتی بیشتر

آموزش گام به گام

  1. گزینه 3 را انتخاب کنید تا تنظیمات DNS به حالت پیش‌فرض بازگردد.
  2. گزینه 1 را انتخاب کنید تا DNSهای هوشمند ست شوند.
  3. از سایر گزینه‌ها برای تست، بررسی یا استفاده از نسخه گرافیکی ابزار استفاده کنید.

در صورت باز نشدن برخی سایت‌ها:

  • مرورگر را کامل ببندید و دوباره باز کنید.
  • کش مرورگر را پاک کنید.
  • مراحل بالا را مجدد تکرار کنید.

اجرای ابزار

فایل run-dns-scripts.bat را اجرا کنید و یکی از گزینه‌های زیر را انتخاب کنید:

1. Set Custom DNS Servers        → تنظیم DNS هوشمند
2. Verify DNS Settings           → بررسی وضعیت تنظیمات DNS
3. Restore Default Settings      → بازگردانی تنظیمات پیش‌فرض
4. Test DNS Servers with URL     → تست DNS با آدرس دلخواه
5. Open DNS Configuration GUI    → باز کردن ابزار گرافیکی
6. Exit                          → خروج
